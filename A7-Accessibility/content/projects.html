Projects

As an aspiring game developer and game designer, I have worked on several private server revival projects within the community for the now defunct MMO game, Toontown Online.

Toontown: Corporate Clash

Position: Game Programmer, Content Designer

Duration: May 2020 - April 2021, March 2022 - Present


While working on the project, as part of the Game Programmer role, I was responsible for developing major features for future content updates in Python using the Panda3D game engine. Features and projects that I worked on under this role include:

  • Major upgrades and refactors of the game's turn-based battle system
  • Taking storyboards and cutscene specifications from the project's Creative, Writing, and Content Design departments and building them in-engine using the project's Cutscene Editor
  • Implementation of various battle designs as part of the v1.3.0 "Hires & Heroes" update
  • Implementation of many various backend systems, quality of life features, and other miscellaneous features required for the release of the v1.3.0 "Hires & Heroes" update
  • Implementation and technical foundation of the real-time C.L.O. battle released in the v1.1.0 "Under New Management" update

As part of the Content Designer role, I was also responsible for collecting, organizing, and presenting user feedback, designing new turn-based battle encounters, referring to and reflecting upon design decisions present in the origin game, Toontown Online, and acting upon user feedback to improve the overall user experience for our players.
